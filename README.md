## 🚀 Getting Started

To view this portfolio locally:  

1. Clone this repository:  
```
   git clone https://github.com/Larrry85e/larrryportfolio.git
```   
2. Go to correct directory:  
```
   cd larrryportfolio
```  
3. Start srver:  
```
   python3 -m http.server
```  
4. Open page in web browser:  
```
   http://localhost:8000/
```
